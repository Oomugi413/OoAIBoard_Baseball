# Baseball Scoreboard

ローカルで動かす野球スコアボードアプリです。

## Font Licenses

このプロジェクトは、スコアボード表示の見た目を端末間で揃えるため、以下のフォントを使用します。

- D-DIN PRO / D-DIN PRO Condensed
  - Source: https://github.com/CyberFei/D-DIN-PRO
  - License: SIL Open Font License 1.1
  - Files: `src/client/fonts/d-din-pro/`
- Noto Sans JP
  - Source: https://fonts.google.com/noto/specimen/Noto+Sans+JP
  - License: SIL Open Font License 1.1
  - Files: `src/client/fonts/noto-sans-jp/`

フォントファイルを差し替える場合は、再配布可能なライセンスであることを確認し、この欄に出典とライセンスを追記してください。
